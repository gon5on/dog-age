●passなど

pass:dogdog

alias pass:dogdog


●aabからapksを作る

※bundletoolのjarファイルをDLしておく
※javaにpathを通しておく

cd /path/to/dog-age
cd etc/key

java -jar bundletool-all-0.9.0.jar build-apks --bundle=../../src/app/release/app.aab --output=../../src/app/release/app.apks --ks=dogage.keystore --ks-pass=pass:dogdog  --ks-key-alias=dogage --key-pass=pass:dogdog


●apksをインストール

※ANDROID_HOMEは環境変数に設定してあるのになんかうまくいかない…
※環境変数を設定したいので、ターミナルはコマンドプロンプトじゃないとうまくいかない…

cd /path/to/dog-age
cd etc/key

set ANDROID_HOME=C:\Android
java -jar bundletool-all-0.9.0.jar install-apks --apks=../../src/app/release/app.apks
